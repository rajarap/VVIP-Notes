To push to git
-------------------------
git init - to initialize a normal maven directory to a git repository directory
git remote add origin <git directory path > - register your local git repo with the actual repo on git
git status
git add .
git status
git commit -m "added minor changes"
git push origin  master   //provide username and password

To pull from git
----------------------
create a folder in your local machine, start cmd from the current folder and give the folowing command.
git clone <git directory path >  //before pulling you have to clone once in your local machine.
git pull  origin master
